# Hide and Seek Map

![Screenshot of Hide and Seek Map](./public/screenshot.png)

A tool for viewing hiding possibilities in Hide and Seek. The map automatically eliminates possible hiding areas based on your questions, helping you narrow down the location of the hiders!
